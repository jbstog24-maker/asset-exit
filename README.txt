# Asset Exit Network Landing Page

## What is included
- `index.html` — the landing page
- `logo.png` — add your real logo with this exact filename, in the same folder

## What to change before publishing
Open `index.html` and replace:

1. Business name:
   - Current: Asset Exit Network

2. Formspree form endpoint:
   - Current: https://formspree.io/f/YOUR_FORM_ID
   - Replace with your real Formspree endpoint, like:
     https://formspree.io/f/abcxyz

3. Company review email:
   - Set the recipient email inside Formspree to the company email you are forwarding leads to.
   - Do not rely only on the placeholder in the code.

## Vercel deploy steps
1. Create a GitHub repo.
2. Upload `index.html` and `logo.png`.
3. Go to Vercel.
4. Import the GitHub repo.
5. Deploy.
6. Test the form.
